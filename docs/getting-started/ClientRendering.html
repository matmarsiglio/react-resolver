Client Rendering

3. Render on the Client

Again, if we're only rendering on the client, we can render like normal:

import React from "react";

React.render(<Router ... />, document.getElementById("app"));

4. Resolve on the Client

If you have a universal application & render on the server, React Resolver handles bootstrapping server-rendered markup via Resolver.render instead of React.render:

import { Resolver } from "react-resolver";

Resolver.render(
  () => <Router ... />,
  document.getElementById("app")
);
